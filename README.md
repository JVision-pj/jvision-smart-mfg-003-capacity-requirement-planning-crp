# 產能規劃洞察站

中長期產能與資源需求規劃工具，銜接業務預測與生產排程。

- 原始系統：產能規劃系統（Capacity Requirement Planning, CRP）
- 產業分類：智慧製造｜製造管理
- 使用部門：生管、廠務、財務、業務
- 工作流程：需求預測輸入 → 產能負荷計算 → 缺口分析 → 因應方案模擬 → 決策輸出
- AI 應用：AI預測季節性產能需求波動、智慧建議最適產能配置方案
- 主要 KPI：產能利用率、缺工/缺產能天數、加班比率
- 代表產品：多為ERP模組延伸功能（如SAP PP、Oracle Fusion），少數獨立品牌為Infor CloudSuite
